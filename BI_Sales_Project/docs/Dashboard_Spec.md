# Dashboard Specification

## KPIs
- TotalSales
- TotalOrders
- TotalCustomers
- AvgOrderValue
- AvgDiscount

## Charts
1. Line: Sales over time (day/month/year).
2. Bar: Top products by sales.
3. Bar: Top customers by sales.
4. Pie: Sales by territory.
5. Bar: Sales by ship method.

## Filters
- Date range (from/to)
- Top N
- Granularity
